# java-student-management-system
Simple Student Management System using Java, JDBC and MySQL

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Features
- Connects Java to MySQL database
- Retrieves student data from the students table
- Displays student details in the console

## Database
Database Name: student_management

Table: students
Columns:
- student_id
- name
- age
- course
- email

## How to Run
1. Install MySQL
2. Create database: student_management
3. Create the students table
4. Update username and password in the Java code
5. Run the program
